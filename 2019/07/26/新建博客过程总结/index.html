新建博客过程总结

By Anonymous; Published on 2019-07-26

##前言

折腾了接近2小时，终于把博客搭建好了((^_^)),当然在搭建的过程中也是遇到了不少的问题，
本想全部记下来的，但是中途不小心手贱关了一次cmd窗口=。=，好多错误和解决方法都没了，没关系，
剩下的还是可以记一下的…

##没进入目标文件夹

Commands:
help Get help on a command. init Create a new Hexo folder. version Display version information.
Global Options:
–config Specify config file instead of using _config.yml
–cwd Specify the CWD
–debug Display all verbose messages in the terminal
–draft Display draft posts –safe Disable all plugins and scripts
–silent Hide output on console
For more help, you can use ‘hexo help [command]’ for the detailed information
or you can check the docs: http://hexo.io/docs/

原因：没进入目标文件夹（部署博客时新建的那个文件夹），我的是blog。
解决：cd blog

##hexo init出错

FATAL ~\blog\ not empty, please run hexo init on an empty folder and then copy your files into it
FATAL Something’s wrong. Maybe you can find the solution here: https://hexo.io/docs/troubleshooting.html

原因：目标文件夹不为空
解决：hexo init 空文件夹名称
cd 文件夹名称
继续操作

##hexo init时长时间不动，停留在某一处

原因：网络问题
解决：科学上网或着傻傻的等=。=

##ERROR Deployer not found: git

原因：未知
解决：npm install hexo-deployer-git –save

##hexo g出错，未配置身份

如果在上传时失败并出现 please tell me who you are 类似子句，代表没有在git中配置身份
解决：右键->Git Bash Here gitconfig->$ git config
$ git config –global user.name “xxx 配置名称
$ git config –global user.eamil “xxx@xxx.com“ 配置邮箱
完成后可以用 $ git config -l(小写L)来检查是否成功

##重温配置部署博客流程

还记得的错误就上面那些了，为什么要手贱啊=。=，接下来总结一下具体流程

0、打开cmd窗口（win键+X，点击运行）

1、搭建环境
1）、去官网下载node.js 网址：https://nodejs.org/en/，完成后可在cmd窗口中用 node -v检查版本号
2）、去官网下载git 网址：https://git-scm.com/downloads，完成后鼠标单击右键会出现 Git GUI HERE 和 Git BASH HERE
3）、在cmd窗口中输入 npm install -g cnpm -registry=https://registry.npm.taobao.org 安装淘宝镜像，之后一样用-v检查是否成功安装
4）、在cmd窗口中输入 cnpm install -g hexo-cli

2、创建博客
1）、在cmd窗口中输入 hexo init blog 创建博客文件夹（默认地址在C:\Users\你自己的用户名）文件夹中
2）、在cmd窗口中输入 hexo s 运行本地博客，此时在浏览器地址中输入 localhost:4000应会出现博客默认主页，确认无误后在cmd窗口Ctrl+C关闭博客
3）、在cmd窗口中输入 hexo n “文章名称” 新建第一篇文章 文章地址在blog\source_posts中，可以用Notepad++来进行编辑
编辑完成后再次启动本地博客，便可在主页中找到该文章

3、部署博客
1）、登录github，点击右上角的New Repository新建博客仓库，命名格式一定要为：自己英文名称（最好和注册账时的名称一致）.github.io
2）、返回cmd窗口输入 cnpm install –save hexo-deployer-git安装插件
3）、用Notepad++编辑blog文件夹中的config.yml文件，在最下方的#deploy标签中新增，type: git，repo: https://github.com/chengchenhe/chengchenhe.github.io.git，branch: master
三个属性，在#Site中添加 email: 自己的邮箱 属性，注意冒号后要加一个空格
4）、在cmd窗口中输入 hexo g 将文件部署到远端，部署完成后在github仓库中会有新建文章内容，从此以后在浏览器地址输入上面注册时的地址便可直接访问博客
5）、完成庆祝一下呗(*^^*)
